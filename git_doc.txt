
※ターミナルで以下を打つ時に $ は要らない

Setup git
$ git init

Commit flow
$ git add file_name
$ git commit -m "Write some commit message"

Check file status
$ git status

Check commit log
$ git log

Check current branch, Existing branch
$ git branch

Create new branch
$ git branch branch_name

Switch branch
$ git checkout add_headline
